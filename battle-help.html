Step 1/3: Allowing socket

In order for it to access the internet, you must make a small modification to your SMBX2 install. Locate your install's main.lua file (SMBX2/data/scripts/base/engine/main.lua)...

Open it with any text or code editor, go to the very end of the file, and add a line with this:

customPackage.loaded["socket.core"] = require("socket.core")

Only ever do this if you truly trust the episode author! Then, you can restart the episode and it should've taken effect. Now when choosing the 'online' option, it should give you a few options.


Step 2/3: Setting up a VPN

By default, it only works via a LAN connection. However, you can use certain programs (such as Hamachi or Radmin VPN) to simulate one over the internet.

You can find guides on how to use them elsewhere, but once you have them setup for everyone you'll be playing with, you can move onto the next bit.
